Kotlin Convertisseur

A small Kotlin app that can:

Convert TND → EUR

Convert Fahrenheit ↔ Celsius

How it works :

 Enter a number

  Click the convert button

  The result shows instantly

Tools:

  Kotlin

  Android Studio

Run the app :
git clone https://github.com/eyaaaaamb/Kotlin-Convertisseur.git


Open it in Android Studio and run.
